# Known Failure Modes

## Pottery
- Handles and spouts may merge into the body.
- Rim/neck/shoulder bands assume upright orientation.

## Figurines
- Broken or occluded limbs may be mislabeled.
- Face/head distinction is approximate in the geometric engine.

## Lithics
- Dorsal vs ventral face and retouch usually need stronger semantic models.

## Metalwork
- Corrosion/patina may not be geometrically separable.

## Organic
- Bone/wood/shell classes vary strongly by material and object type.


## Force-all mode

Force-all taxonomy segmentation is for debugging only. It deliberately attempts every possible part and can create false labels on fragments. Do not use force-all outputs as clean training data for Tool 3 without review.
